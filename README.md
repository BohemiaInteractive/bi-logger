Logger module
=============

Adds ability to log errors and unhandler exceptions into files located in `/logs/` directory.

How to use
----------

Adding into project (`package.json`):
```json
{
  "dependencies": {
    "logger": "git+ssh://git@webdev-git.czupc.bistudio.com:juraj/logger.git"
  }
}
```

Example use:
```js
    //...something
```
